Options

options description
transition "fade", "crossfade", "slide", "swap"
default: fade
interval Amount of time in between each "twinkle"
default: 2000
speed Speed of "twinkle"
default: 1000
easingMethod Easing method for "twinkle" as found at http://gsgd.co.uk/sandbox/jquery/easing/
default: "easeOutQuint"
hidden An array of image sources. This array is the set of images that are initially hidden.
default: [ ]
direction Direction for "slide" and "swap" transitions. If no direction is provided, twinkle.js will choose random directions.
default: none

Example


$('#container').twinkle({
  transition : 'slide',
  interval : 3000,
  speed : 1200,
  easingMethod : 'easeOutQuint',
  direction : 'left',
  hidden : [
    'path_to_image.jpg',
    'path_to_image.jpg',
    'path_to_image.jpg',
    'path_to_image.jpg'
  ]
});

Demo

$('.fade').twinkle({

Crossfade


$('.crossfade').twinkle({
  hidden: [ARRAY],
  transition: 'crossfade'
});

Slide


$('.slide').twinkle({
  hidden: [ARRAY],
  transition: 'slide',
  speed: 1400
});

Swap


$('.swap').twinkle({
  hidden: [ARRAY],
  transition: 'swap'
});

Locking

If you want to "lock" an image in the grid, just give it's parent div a class of locked.
This particular image will always remain static. Feel free to apply this class to as many
images as you please. 


<div class="locked">
  <img src="image_path.jpg">
</div>
